> JUnit을 활용한 테스트 코드 작성

- Pawn 테스트 클래스
    - Pawn 생성 테스트 메소드 추가

- Pawn 생성 테스트 메소드 리팩토링

## step 2 체스판 생성

> JCF를 이용한 객체 관리
